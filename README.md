# NAWIN KUMAR US - Portfolio Website

A professional, recruiter-friendly portfolio website built with HTML, CSS, and JavaScript.

## Quick Start

1. **Add your profile picture**: Place a professional photo named `profile.jpg` in this folder. If no image is found, a placeholder with your initials will display automatically.

2. **Add your resume**: Place your resume PDF file named `resume.pdf` in this folder for the Download Resume button to work.

3. **Open the website**: Double-click `index.html` or open it in a browser.

## Customization

- **Profile Picture**: Add `profile.jpg` (recommended: 400x400px or larger, square format)
- **Resume**: Add `resume.pdf` for the download button
- **Contact Form**: The form currently shows a success message. To enable actual email sending, integrate with a service like Formspree, EmailJS, or your backend.

## Features

- Clean, professional corporate design
- Fully responsive (mobile, tablet, desktop)
- Smooth scrolling navigation
- Sticky header with active section highlight
- Typing effect on hero section
- Scroll-triggered animations
- ATS-friendly structure

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
